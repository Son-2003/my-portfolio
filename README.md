# 💼 My Portfolio

This is my personal developer portfolio website, built using **React**, **Tailwind CSS**, and **TypeScript**.  
It features my projects, skills, and a contact form that sends messages directly to my email via **EmailJS**.

## 🔧 Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- Deployed on Vercel
